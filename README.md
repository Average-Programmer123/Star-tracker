🌌 Ever wondered how people click those wonderful images of the sky? Well, you don't need any fancy setup just a mobile phone and 
a few electronics is more than enough!

Here is a list of all the items you need:
1. 💻Laptop with a Python editor and Numpy, Scipy, Pyserial installed
2. 🔌Arduino Uno/Nano
3. ⚙️Stepper motor (28BYJ-48)
4. 📱Mobile phone with adjustable shutter speed
5. 🔌USB cable
6. ⚡️Power bank
7. 🏎Chassis
